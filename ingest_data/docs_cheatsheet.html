QRT.C3 Grand Final — Cheat sheet

Pre-flight checks

  • Dashboard at https://c3.qwerty.technology/dashboard
  • Have your personal game token for Stage 2 (playing with bots)
  • Have your team game token for Stage 3 (live competition)
  • Have codex running with sufficient credits

How the game works

  • Each round, a payoff matrix is sampled and a prediction is proposed, e.g. R>=* or P>R:
 ?   -2   -3
-1   -3   +3
+1    0   -1
  • You have 5s to opt in and agree or disagree with the prediction, or abstain.
  • You can broadcast your intentions publicly if you wish.
  • You play simultaneously against all other opponents.

How to do well and score points

  • Use LLM agents (codex) extensively: to write code, brainstorm, analyse data, etc. A good starting point is to feed the game documentation and URL to codex.
  • Leverage teamwork: strategise together, split tasks, share codebase effectively (e.g. git repo).
  • Make sure you switch from individual game token to team token by the end of Stage 2.
  • Score points in play: use the best action given the payoff matrix and your opponents. +5 for opt-in.
  • Score points by betting: chip in 15 when entering the pot; losers forfeit pot to winners. Know when to bet & which side to pick.
  • Avoid penalties: make sure you are online if you opt in. A missed action costs −1 per opponent.
